# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-06

### Added
- Initial release of the media-tools plugin with two skills
- **pretext** skill: Pretext text measurement library integration guide with API reference and patterns
- **youtube-downloader** skill: Interactive YouTube video downloading with resolution and subtitle selection
- Cross-platform support: Claude Code, Claude Desktop, Cursor, Manus, ChatGPT, Codex CLI
- Claude Code plugin marketplace support
- Cursor plugin support
- GitHub Release workflow for zip distribution
